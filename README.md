# word learning models

Christine Soh Yue, Sandy LaTourrette, Charles Yang, and John Trueswell

## model_code

`model_code` is a folder containing the computational models.

### models

`models` contains the code for the MBP model (`memory_bound_pursuit.py`), the Pursuit model (`pursuit_learner.py`), and the Familiarity Uncertainty based Global model (`kachergis.Rmd`), adopted from George Kachergis' 
(github)[https://github.com/kachergis/word_learning_models/tree/master].

### experiment_code

`experiment_code` contains the files to run the various simulations and experiments with the MBP and Pursuit models. The FUbG model is all run within the Rmd code.

### data
`data` is a folder containing the data txt files.
